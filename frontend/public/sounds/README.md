# Sound effects

Place these files in this directory for the sound effects system:

| File | Description | Duration |
|------|-------------|----------|
| `pencil-scratch.mp3` | Soft graphite on paper (ThinkingDrawer expand) | ~0.3s |
| `typewriter.mp3` | Delicate mechanical click (citation/source drawer) | ~0.2s |
| `search-whoosh.mp3` | Subtle air movement (search start) | ~0.4s |
| `chime.mp3` | Very soft bell (optional, e.g. response complete) | ~0.5s |
| `tap.mp3` | Gentle click (button tap) | ~0.1s |

**Free sources:** [Freesound.org](https://freesound.org), [Zapsplat.com](https://www.zapsplat.com), [Mixkit.co](https://mixkit.co/free-sound-effects/)

If a file is missing, the app will still run; `playSound()` will no-op and log a warning in the console.
